[SAP] Como criar uma FIELD-EXIT

0

Volta e meia aparece a necessidade de criar uma consistência adicional para um campo no SAP. As vezes surge também a necessidade de criar gatilhos para execução de processos baseados no preenchimento de um campo. Como diria alguém com experiência na área de desenvolvimento de sistemas, "There is no silver bullet" - não há bala de prata. Mas existem recursos dentro do SAP que permitem criar essas consistências. Este recurso é a FIELD-EXIT: uma forma de programar em cima do preenchimento de um campo da tela.

Neste post, vou mostrar como configurar e programar uma FIELD-EXIT. E você verá que, assim como muitos outros recursos do SAP, é é uma tarefa simples.
